# Vue 가이드
## 1. Vue 설치
```bash
npm install vue@3.2.13
```

## 2. Vue Router 설치
```bash
npm install vue-router
```

## 3. ESlint 가 난리라면
```bash
npm uninstall eslint eslint-plugin-vue
npm install eslint@^8.0.0 eslint-plugin-vue@^9.0.0 --save-dev
```
